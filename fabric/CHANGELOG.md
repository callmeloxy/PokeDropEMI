# PokeDropEMI 1.0.0

## Summary

This update improves how contextual drop conditions are displayed in the PokeDrops EMI category, without adding configuration or changing the core behavior of the mod.

## Improved

- Improved display of conditions from `spawn_pool_world`
- Technical conditions are now converted into more readable text
- Improved dimension condition text:
  - Nether
  - End
  - Overworld
- Improved weather condition text:
  - rain
  - thunder
  - clear weather
- Improved time condition text:
  - day
  - night
  - dawn
  - dusk
- Improved text for some periods, moon phases, and structures
- Cleaned up technical tags to avoid raw-looking tooltips
- Kept the current smart drop merging behavior

## Notes

- No configuration was added in this version
- Server-only datapacks still cannot be read client-side without a network sync system
